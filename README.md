# 📊 Facebook Campaign Performance Analysis

## 📌 Project Goal
The goal of this project is to analyze Facebook campaign performance data to evaluate ad spend efficiency, audience engagement, targeting effectiveness, and campaign ROI, enabling better marketing budget allocation and audience optimization decisions.

---

## 📌 Business Problem
The business wants to answer:

- Which campaigns generate the highest CTR?
- Which campaigns have the lowest CPI and CPM?
- Which audience age groups engage the most?
- How efficient is the reach → impressions → interactions funnel?
- Where should ad budget be reallocated?

---

## 📂 Dataset
The dataset contains Facebook marketing campaign performance data including:

- Campaign Name
- Cost
- Reach
- Impressions
- Interactions
- CTR
- CPI
- CPM
- Age Group
- Frequency
- Engagement Rate

This project uses campaign-level and audience-level ad performance data.

---

## 🛠 Tools Used
- SQL → KPI calculations and campaign-level data preparation
- Tableau → Dashboard design and storytelling
- Excel / CSV → Raw data source
- GitHub → Documentation and portfolio showcase

---

## 🔄 Workflow
1. Imported campaign data into SQL
2. Cleaned and validated campaign metrics
3. Built KPI calculations:
   - CTR
   - CPI
   - CPM
   - Engagement Rate
   - Frequency
4. Created campaign performance summary datasets
5. Built 3 Tableau dashboards:
   - Overall Campaign Performance
   - Audience Targeting vs Engagement
   - Efficiency Analysis
6. Added interactive filters and parameter-based decision view

---

## 📊 KPI Metrics
- CTR
- CPI
- CPM
- Engagement Rate
- Frequency
- Funnel Conversion
- Audience Response
- Best Campaign Identification

---

## 📈 Key Insights
- Certain campaigns delivered high CTR with low CPI, making them highly cost-efficient
- Younger age groups showed stronger engagement response
- Funnel drop-off highlighted scope for better audience targeting
- Some campaigns had high frequency but lower engagement, indicating ad fatigue
- Best campaigns combined low CPI with strong CTR performance

---

## 💡 Business Recommendations
- Reallocate budget toward high CTR + low CPI campaigns
- Reduce spend on high-frequency low-engagement campaigns
- Improve targeting for weak funnel drop-off stages
- Prioritize top-performing audience age groups
- Continuously monitor campaign fatigue through frequency KPI
